# NullDox - 虚假开盒插件

## 📖 简介

NullDox 是一个用于 AstrBot 的娱乐插件，通过生成虚构的用户信息来模拟"开盒"效果。**所有数据均为随机生成，不涉及任何真实信息泄露**。

> ⚠️ **免责声明**：本插件仅供娱乐用途，所有生成的信息均为虚假数据。请勿用于任何非法或恶意目的，使用者需自行承担一切法律责任。

## ✨ 功能特性

- **主动查询**：通过 `盒` 命令查询指定 QQ 号，生成虚假身份信息
- **自动触发**：检测到群成员主动退群时，自动生成退群成员的虚假信息
- **真实感增强**：生成包含手机号、IP地址、详细地理位置的完整信息
- **头像展示**：配合 QQ 头像图片，增加视觉效果
- **地理数据库**：内置中国行政区划数据，生成真实的地址信息

## 📖 使用说明

### 主动查询

发送命令 `/盒 [QQ号]` 或者 `/盒 [@用户]` 即可生成该账号的虚假信息：

```
/盒 123456789
```

**返回示例**：
```
🔍 身份检索完毕
🆔 账号：123456789
📱 手机：13812345678
🌐 IP地址：119.75.218.77
📍 物理地址：四川省成都市武侯区天府大道
```

同时会返回该 QQ 号的头像图片。

### 自动触发

当有群成员主动退群时，插件会自动检测并生成退群成员的虚假信息：

```
🚨 检测到 123456789 退出群聊,正在进行盒打击
🔍 身份检索完毕
🆔 账号：123456789
🚪 退出群聊：987654321
📱 手机：15987654321
🌐 IP地址：58.246.137.88
📍 物理地址：北京市朝阳区建国路
```

## ⚙️ 配置说明

本插件无需额外配置，开箱即用。

## 📊 数据生成规则

| 数据类型 | 生成规则 |
|---------|---------|
| **手机号** | 随机选择常用号段 + 8位随机数字 |
| **IP地址** | 随机生成常用公网IP段（排除私有地址和保留地址） |
| **地理位置** | 从内置行政区划数据库中随机选择 |
| **QQ头像** | 使用 qlogo.cn API 获取真实头像 |

## 🛠️ 技术实现

- **事件过滤**：自定义 `DecreaseTypeFilter` 过滤主动退群事件
- **地理位置扁平化**：将嵌套的 JSON 数据展平为地址字符串列表
- **异步支持**：完全支持异步操作，不影响机器人性能
- **错误处理**：完善的数据验证和异常捕获机制

## ❓ 常见问题

### Q: 退群事件为什么不触发？
A: 请确保：
- 使用的是 AIOCQHTTP 协议适配器
- 机器人有接收群通知的权限
- 退群类型为 `leave`（主动退群），而非被踢出
